# AMO Brand & Architecture Guidelines

## Tech Stack
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Hosting: Vercel
- Backend: None (Zero-cost architecture)
- Email API: Loops.so

## Visual System (UI/UX Pro Max)
- **Base Theme:** Deep, rich dark mode using `bg-zinc-950`.
- **Background:** DO NOT use flat colors; implement subtle, glowing radial gradients in purple/cyan at low opacity to feel cinematic.
- **Brand Colors:** Burgundy Red (`#842935`) for primary accents/buttons, and White (`#FFFFFF`).
- **Glassmorphism:** The waitlist card MUST use `bg-white/5 backdrop-blur-xl border border-white/10 shadow-2xl`.
- **Typography:** Use `Inter` or `system-ui`.
- **Headings:** Very large, bold, and tracking-tight.
- **Body:** Relaxed spacing, slightly muted for readability.

## Vibe & Positioning
- Premium, futuristic, minimal, and confident.
- Not a snack, not a protein bar; a modern, premium meal solution for busy schedules.
